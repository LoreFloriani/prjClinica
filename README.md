# PrjClinica

PrjClinica e una semplice applicazione web per la gestione dei pazienti di una clinica veterinaria.

Permette di visualizzare gli animali ricoverati, aggiungerne di nuovi, modificare il loro stato di salute e consultare gli animali dimessi. Il progetto usa Node.js, Express, MongoDB e Multer per il caricamento delle immagini.
